## 7-post thread draft

### 1/7
처음엔 Windows에 `Win+V`라는 기능이 있는지도 몰랐습니다.
온라인 서류 등록 업무를 하다가 우연히 알게 됐고, 정말 "유레카"를 외쳤어요.

### 2/7
기본 멀티 클립보드는 생각보다 훨씬 편했습니다.
최근에 복사한 항목을 다시 찾을 수 있으니까 반복 복붙이 빨라졌거든요.

### 3/7
그런데 실무에서는 여기서도 부족했습니다.
날짜, 담당자, 이벤트명처럼 조금씩 바뀌는 텍스트 묶음을 계속 번갈아 써야 했어요.

### 4/7
즉, 최근 복사 목록만 보는 게 아니라
`관련된 텍스트 3개를 하나의 세트로 저장하고 빠르게 바꿔 쓰는 기능`
이 더 필요했습니다.

### 5/7
그래서 만든 게 `ClipStack`입니다.
Windows 클립보드 히스토리(Win+V)를 실제 업무 흐름에 맞게 확장하는 작은 유틸리티예요.
클립을 저장하는 것보다, 다시 꺼내 쓰는 흐름을 덜 끊기게 만드는 데 더 집중했습니다.

### 6/7
핵심은 단순합니다.
- 3개 항목을 세트로 저장
- Win+V에 다시 주입
- Ctrl+Alt+1~6으로 즉시 전환

### 7/7
반복 입력 작업이 많은 분들께 도움이 되면 좋겠습니다.
