# 클리앙/커뮤니티 게시글 완성본 (KO)

## 제목 후보
- 윈도우 Win+V를 실무형으로 확장한 작은 유틸 하나 만들어봤습니다
- 반복 입력 작업이 많아서 만든 클립보드 세트 유틸, ClipStack
- Win+V는 좋은데 실무에서는 조금 아쉬워서 만든 도구입니다

## 본문
안녕하세요.
평소 반복적인 입력 작업을 하다가 불편해서, 주말 동안 작은 유틸리티를 하나 만들어봤습니다.

이름은 **ClipStack**이고,
Windows 기본 클립보드 히스토리인 `Win+V`를 실제 업무 흐름에 조금 더 맞게 확장하는 도구입니다.

제가 불편했던 지점은 단순히 복사 붙여넣기 자체보다,
복사한 내용이 많아졌을 때 다시 찾아 쓰는 과정이 자꾸 흐름을 끊는다는 점이었습니다.
특히 날짜, 담당자, 사번, 이벤트명처럼 조금씩 바뀌는 항목들을 반복해서 넣을 때 더 그랬습니다.

그래서 자주 쓰는 텍스트 3개를 하나의 세트처럼 저장해두고,
필요할 때 다시 Windows 클립보드 히스토리에 넣고,
단축키로 빠르게 전환할 수 있게 만들어봤습니다.

핵심은 이런 흐름입니다.
- 3개 항목 세트 저장
- Win+V로 다시 주입
- Ctrl+Alt+1~6으로 세트 전환
- Ctrl+Alt+0으로 관리창 열기

무거운 기능은 최대한 빼고,
반복 입력 작업에서 바로 도움이 되는 흐름에 집중했습니다.

README 첫부분에 경험담과 GIF 데모도 넣어두었습니다.
한 번 보면 어떤 문제를 해결하려는지 바로 이해되실 거예요.

- GitHub: https://github.com/ahnsungkwon/ClipStack

혹시 저처럼
- 등록 업무
- 운영/사무 입력
- 반복 문구 입력
- 템플릿 복붙
같은 작업을 자주 하시는 분이 있다면,
써보시고 어떤 흐름에서 특히 쓸 만한지 피드백 주시면 감사하겠습니다.

광고처럼 보이기보다,
비슷한 불편이 있었던 분들에게 도움이 될까 해서 공유드립니다.
