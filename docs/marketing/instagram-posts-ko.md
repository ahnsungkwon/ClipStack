## Post 1 - 공감형
아직도 반복 입력할 때 한 줄씩 복사하고 붙여넣고 있나요?

저도 예전에는 그랬습니다.
그러다 Windows 기본 기능인 `Win+V` 멀티클립보드를 알고 정말 편해졌어요.

그런데 실무에서는 날짜, 담당자, 이벤트명처럼 조금씩 바뀌는 내용들을 묶어서 빠르게 바꿔 쓰는 기능이 더 필요했습니다.

그래서 만든 게 ClipStack입니다.

반복 복붙을 줄이고,
자주 쓰는 텍스트 세트를 빠르게 전환할 수 있게 해주는 도구입니다.

## Post 2 - 문제/해결형
Win+V는 편합니다.
근데 실무에서는 그걸로 끝이 아니더라고요.

최근 복사 목록만 보는 게 아니라,
관련된 3줄을 세트로 저장하고
단축키로 바꿔 쓰고 싶었습니다.

ClipStack은 그 흐름을 만들어줍니다.

Grab → Stack → Sync → Switch.

작은 도구지만,
반복 작업에서는 체감이 큽니다.

#윈도우 #멀티클립보드 #생산성 #사무자동화 #업무툴 #ClipStack

## Post 3 - 제작 스토리형
어떤 도구는 거창한 문제를 해결하지 않습니다.
대신 매일 반복되는 작은 짜증을 없애줍니다.

ClipStack은 그런 도구입니다.

복붙을 많이 해야 하는 실제 업무에서,
`복사한 내용을 다시 찾는 순간`이 너무 자주 흐름을 끊어서 만들었습니다.

필요한 사람이 한 명이라도 더 편해졌으면 좋겠습니다.
README 데모 보고 피드백 주셔도 좋습니다.
