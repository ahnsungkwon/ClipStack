# 뽐뿌용 게시글 완성본 (KO)

## 제목 후보
- Win+V를 조금 더 실무형으로 쓸 수 있게 만든 유틸입니다
- 반복 입력 작업 때문에 만든 작은 클립보드 도구 공유합니다
- 윈도우 클립보드 히스토리 쓰다가 아쉬워서 하나 만들어봤습니다

## 본문
안녕하세요.
반복 입력 작업을 하다가 불편해서 작은 유틸 하나 만들어봤습니다.

이름은 `ClipStack`이고,
Windows 기본 클립보드 히스토리인 `Win+V`를 조금 더 실무적으로 쓸 수 있게 만든 도구입니다.

제가 불편했던 건 복사 자체보다,
복사한 내용이 많아졌을 때 다시 찾고 다시 붙여넣는 과정이었습니다.
특히 날짜, 담당자명, 사번, 이벤트명처럼
조금씩 바뀌는 텍스트를 계속 번갈아 넣어야 할 때 더 불편했습니다.

그래서 자주 쓰는 텍스트 3개를 하나의 세트처럼 저장해두고,
필요할 때 다시 불러와서 쓸 수 있게 만들어봤습니다.

대충 이런 흐름입니다.
- 3개 항목 세트 저장
- Win+V에 다시 넣기
- Ctrl+Alt+1~6으로 세트 전환
- Ctrl+Alt+0으로 관리창 열기

무거운 프로그램 느낌보다는,
반복 작업에서 바로 써먹기 좋은 가벼운 유틸 쪽으로 만들었습니다.
