## 런칭 포스트 1
복붙이 많은 작업은 생각보다 자주 흐름이 끊깁니다.

ClipStack은 클립보드를 “저장”하는 도구라기보다,
이전에 복사한 내용을 다시 꺼내 쓰는 과정을 덜 귀찮게 만들려고 만든 프로젝트입니다.

README 첫부분에 GIF 데모까지 넣어뒀습니다.
보면 바로 이해됩니다.

GitHub:
https://github.com/ahnsungkwon/ClipStack

## 런칭 포스트 2
Win+V를 알고 나서 분명 편해졌는데,
실제 업무에서는 거기서도 조금 부족하더라고요.

특히 날짜, 담당자, 이벤트명처럼
조금씩 바뀌는 텍스트 묶음을 번갈아 써야 할 때요.

그래서 자주 쓰는 3개 항목을 세트처럼 저장하고,
빠르게 다시 꺼내 쓸 수 있게 ClipStack을 만들었습니다.

README 데모 보고 의견 주셔도 좋습니다.
https://github.com/ahnsungkwon/ClipStack
