# YouTube Shorts Scripts (KO)

## Shorts 1 - Win+V를 처음 알았을 때
### Recommended title
윈도우에서 아직도 Ctrl+C, Ctrl+V 하나씩 하세요? (생산성 꿀팁)

### Hook
"아직도 텍스트 하나 복사하고, 창 넘어가서 붙여넣고 계신가요?"

### Script
온라인 등록 업무를 하다 보면
똑같은 내용을 계속 넣어야 할 때가 있죠.

저도 예전에는 몰랐는데,
윈도우에 기본 멀티클립보드가 있습니다.
`Win + V`.

이거 처음 알았을 때 진짜 충격이었어요.
근데 실무에서는 여기서 한 가지가 더 필요했습니다.

관련된 여러 줄을 세트로 저장하고,
필요할 때 바로 바꿔 쓰는 기능.

그래서 만든 게 ClipStack입니다.

### Shot flow
- 0~3초: 답답한 한 줄 복붙 화면
- 4~8초: Grab 단계
- 9~15초: Sync 단계 + Win+V 결과

### End CTA
"무료로 공개해뒀습니다. 고정 댓글이나 설명란에서 확인하세요."

## Shorts 2 - 왜 만들었는지
### Hook
"기본 멀티클립보드만으로는 부족했던 이유"

### Positioning line
`Win+V의 한계 극복`으로 설명하면 더 직관적입니다.

### Script
Win+V는 정말 편합니다.
하지만 실제 업무는
항상 완전히 같은 텍스트만 반복되지 않아요.

날짜가 바뀌고,
담당자가 바뀌고,
이벤트별로 세트가 달라집니다.

그때마다 다시 복사하고,
다시 정리하는 게 너무 번거로웠어요.

그래서 자주 쓰는 3줄을 세트로 저장하고,
단축키로 바로 전환할 수 있게 만들었습니다.

### End CTA
"반복 복붙 업무를 줄이고 싶다면 ClipStack."

## Shorts 3 - 실제 사용 흐름
### Hook
"이건 그냥 복붙 툴이 아닙니다"

### Script
1. 필요한 3줄을 복사합니다.
2. ClipStack에 세트로 저장합니다.
3. 세트를 Windows 클립보드 히스토리에 넣습니다.
4. 단축키로 세트를 바로 바꿉니다.

Grab.
Stack.
Sync.
Switch.

복붙이 아니라,
작업 흐름을 바꾸는 방식입니다.

### End CTA
"반복 입력이 많은 사람일수록 체감이 큽니다."
